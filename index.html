      About Canary NPC Creator

      ×

      A modern, web-based visual editor for creating NPCs for Canary OTServ.

      Version v1.0.0
      Canary Version Compatible with latest
      Developed by Eratsu feat. Claude
      Built for Canary OTServ Community

      🎯 Features

      • Visual character editor with 249+ outfits

      • Complete shop system (buy/sell items)

      • Custom conversation messages

      • Keywords system for interactions

      • Real-time character preview

      • Lua script generation and export

      🗄️ Database

      • items.xml loading all items

      • outfits.xml (male/female) with addons

      • mounts.xml with enable/disable toggle

      🔗 Links

      • Canary Repository

      💡 Tips

      • Use New to reset all data

      • Use Open to load existing NPC files

      • Use Generate to export Lua scripts

      • Use Random for random character colors
